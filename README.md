#bu
================================

Utility for quickly backing up files and directories.  Copy into your path and set to execute.  

##How to run it
--------------------------------

```
usage: bu [options] filename (dirname)

 file and diretory backup utility
 
 optional arguments:
   -h, --help            show this help message and exit
   -f files [files ...]  filename(s) to backup
   -D backupdir          backup dir name (default BACKUP) 
```
